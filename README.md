# Personal Expense Tracker (DBMS Mini Project)

## Description
This is a DBMS mini project developed as part of college coursework. The project is designed to store and manage personal income and expenses using a structured database.

It demonstrates basic database concepts such as table creation, relationships, and SQL queries.

---

## Technologies Used
- MySQL  
- SQL  

---

## Project Structure

- `sql/` → SQL files (tables, data, queries)  
- `outputs/` → screenshots of execution  
- `diagram/` → ER diagram  
- `report.pdf` → project report  

---

## Features
- Store income and expenses  
- Categorize transactions  
- View expense details  
- Calculate remaining balance
